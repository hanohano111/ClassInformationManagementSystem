### 2.2 详细用例

#### 教师用例

```mermaid
left to right direction
actor 教师

rectangle 班级管理 {
    usecase "创建班级" as C1
    usecase "修改班级" as C2
    usecase "解散班级" as C3
    usecase "查看班级列表" as C4
}

rectangle 成员管理 {
    usecase "添加成员" as M1
    usecase "移除成员" as M2
    usecase "设置管理员" as M3
}

rectangle 通知管理 {
    usecase "发布通知" as N1
    usecase "删除通知" as N2
}

rectangle 作业管理 {
    usecase "发布作业" as A1
    usecase "删除作业" as A2
    usecase "批改作业" as A3
}

rectangle 请假管理 {
    usecase "审批请假" as L1
}

rectangle 签到管理 {
    usecase "发起签到" as S1
    usecase "查看签到记录" as S2
}

教师 -- C1
教师 -- C2
教师 -- C3
教师 -- C4
教师 -- M1
教师 -- M2
教师 -- M3
教师 -- N1
教师 -- N2
教师 -- A1
教师 -- A2
教师 -- A3
教师 -- L1
教师 -- S1
教师 -- S2
```
