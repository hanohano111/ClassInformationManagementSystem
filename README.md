# 班级信息管理小程序

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ 项目简介

班级信息管理小程序是一款专为师生打造的一站式班级事务管理平台。它致力于简化班级内的信息流转、教学互动和日常管理，功能涵盖班级创建、通知发布、作业管理、在线签到、请假审批以及师生即时通讯等。借助微信云开发，本项目可被快速部署和应用，为教育场景提供高效、便捷的数字化解决方案。

## 🚀 主要功能

- **🏫 班级管理**:
    - 教师可轻松创建、编辑和解散班级。
    - 通过唯一的班级邀请码，学生可以安全、便捷地加入班级。
    - 支持设置多位管理员，协助管理班级事务。
- **📢 通知与作业**:
    - 教师可随时发布班级通知，确保信息及时触达每一位成员。
    - 支持发布、批改作业，学生可在线提交并查看反馈。
- **🗓️ 请假与签到**:
    - 学生可在线提交请假申请，教师在移动端即可完成审批。
    - 提供**位置签到**和**签到码**两种模式，满足不同场景下的考勤需求。
    - 自动统计出勤数据，方便教师掌握情况。
- **💬 即时沟通**:
    - 内置班级群聊功能，方便师生随时随地进行交流。
    - 未读消息提醒，确保重要信息不错过。
- **👤 个人中心**:
    - 用户可以方便地编辑个人资料。
    - 聚合常用功能入口，提供个性化服务。

## 🛠️ 技术栈

- **前端**: 微信小程序原生开发 + [TDesign Weapp](https://tdesign.tencent.com/miniprogram/) 组件库
- **后端**: 微信云开发 (云函数、云数据库、云存储)
- **代码质量**: ESLint + Prettier

## 快速上手

### 环境准备

- **微信开发者工具**: 最新版本，并开通云开发功能。
- **Node.js**: `v16.x` 或更高版本。
- **微信小程序 AppID**: 一个已开通云开发的 AppID。

### 部署步骤

1.  **克隆项目到本地**
    ```bash
    git clone <your-repo-url> ClassInformation
    cd ClassInformation
    ```

2.  **安装项目依赖**
    ```bash
    npm install
    ```

3.  **导入项目**
    - 打开微信开发者工具，选择 `导入项目`。
    - 项目目录选择刚刚克隆的 `ClassInformation` 文件夹。
    - 填写你的微信小程序 `AppID`。

4.  **配置云环境**
    - 在 `app.js` 文件的 `wx.cloud.init` 方法中，填入你的云开发环境 `env` ID。
    - 如果没有环境，请先在微信开发者工具的“云开发”控制台中创建一个。

5.  **部署云函数**
    - 在 `cloudfunctions` 目录下，包含了项目所需的所有云函数。
    - **重要**: 逐个进入每个云函数的子目录，执行 `npm install` 安装其独立依赖。
    - 右键点击 `cloudfunctions` 目录，选择 `上传并部署：所有文件`，将所有云函数部署到云端。

6.  **运行和调试**
    - 点击开发者工具的 `编译` 按钮，即可在模拟器中预览。
    - 建议使用真机调试以获得最佳体验，特别是对于位置、扫码等功能。
